Webserv テストページ

http://localhost:8080
サーバーが正常に動作しています！
このページはWebservプロジェクトの動作確認用テストページです。 このページが表示されていれば、HTTPサーバーが正常にHTMLファイルを配信できていることを意味します。

サーバー情報

  • プロジェクト: Webserv
  • 言語: C++ 98
  • プロトコル: HTTP/1.1

Webservの主な機能

  • 複数ポートでのリッスン
  • GET, HEAD(option), POST, DELETEメソッドの対応
  • 設定ファイルによるルーティング
  • ディレクトリリスティング
  • CGIスクリプトの実行GET, POST
  • ファイルアップロード
